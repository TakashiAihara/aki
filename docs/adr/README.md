# Architectural Decision Records

This directory contains Architectural Decision Records (ADRs) for the project.

## How to create a new ADR

1. Copy `template.md` to a new file in this directory, e.g., `0001-record-architecture-decisions.md`.
2. Fill in the template with the details of the architectural decision.
3. Submit a Pull Request.

## Tools

* [MADR](https://adr.github.io/madr/) - The template used here is based on MADR.
